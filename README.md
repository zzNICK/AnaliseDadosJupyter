## Análise de dados em uma empresa

Nessa análise, o programa deverá auxiliar uma empresa de Telecom em seu histórico de clientes dos últimos anos, tendo em vista que
a empresa está com problemas de cancelamento em massa, chegando
ao todo em 26% de cancelamentos dos serviços utilizados pelos usuários.

Nesse sentido, deve ser feito uma análise de dados para
achar o eventual problema das altas demandas de cancelamentos dentro da empresa.
